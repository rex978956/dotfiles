# dotfiles

## Usage

### Initialization

Install Zshell, Tmux, Vim

```shell=
git clone https://github.com/rex978956/dotfiles .dotfiles
cd ~/.dotfiles && chmod +x ./setup.sh && sudo ./setup.sh
```

You maybe should reboot.

### Install tools

Install batcat ,python, powerline, pwndbg

```shell=
cd ~/.dotfiles/Misc
chmod+x * && ./setup.sh
```
